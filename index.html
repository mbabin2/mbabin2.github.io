About me

I am a PhD candidate and lecturer in Western University's department of Computer Science. My research focuses on novel applications of procedural content generation (PCG) for video game level generation, security, and design. I am the instructor for CS4482: Intro to Game Programming. I have been building games and virtual environments professionally over the past 5 years. I am now looking for industry positions as I prepare to graduate in May 2025.

BOLT-OUT (2017): My first full game project was developed as my final submission for CS4482—the very course I now teach. Built in Unity, it serves as a tribute to one of my all-time favorite games, F-ZERO GX. While video games have been a lifelong passion of mine, the process of creating BOLT-OUT solidified my decision to pursue a career in game development. Below, you’ll find a selection of both my research and development experience.

Projects & Research

A Hybrid Approach to Procedural Dungeon Generation (2020)

My master's thesis focused on the procedural generation of dungeon environments, combining multiple AI techniques, including artificial neural networks and N-Gram models, as well as constructive algorithms and context-free grammars. This hybrid approach enables the generator to create millions of unique layouts and over 5,400 distinct puzzles, all generated in real time with guaranteed solvability.

Leveraging RL and WFC for Improved Procedural Level Generation (2021)

Presented at the Foundation of Digital Games (FDG) 2021, this work improved the playability of game levels generated by the wavefunctioncollapse (WFC) algorithm. We trained a reinforcement learning agent to replace WFC’s heuristic, testing the approach on Super Mario Bros. An example level generated with our method is shown, alongside an A*-driven agent navigating it.

Modeling Player Behaviour in 2D Platformers (2021)

A project aimed at creating believable NPC behavior by emulating player actions in a 2D platformer. I trained four agents in total: two using a behavioral cloning algorithm and two using popular reinforcement learning (RL) algorithms. The video above showcases a replay of the human player alongside each agent across 100 procedurally generated levels.

Combating Computer Vision-Based Aim Assist Tools in Competitive Online Games (2023)

Presented at ICEC 2023, this work used procedural generation to counter AI-based aimbots in competitive shooters. We employed an evolutionary strategy to create decoy textures from patches of noise that appear normal to players but are misclassified as targets by ML-based computer vision models. Results show a decoy successfully tricking an aimbot, labelling our object has a player.

World Omnidirectional Maze: WOM (2024)

Solo project for the 2024 LoJam x GDS game jam, themed 'Flipside.' WOM (also WOM when flipped) is an action-adventure game inspired by Tunic, where the world flips to reveal four distinct environments. Players solve puzzles by connecting elements across these worlds to reach the exit. I created all graphics, music, and programming, using only a player model and rigging from an asset pack.

Full playthrough here.

Swarmancer: Path of Bones (2024)

My solo entry for LoJam 2024, Swarmancer: Path of Bones is an action roguelike where players control a swarm of rats to defeat enemies. Unlike my previous solo hackathon project, I focused on experimenting with AI technologies rather than creating all assets myself. The main challenge was optimizing pathfinding for thousands of entities. I solved this by using flow fields for swarm navigation while enemies relied on a more refined navmesh. This project won the Judges' Award for Best Gameplay.

Game Balance Through Procedural Content Generation (2025)

Published at FDG 2025, this work explores training an AI agent to balance games by generating new content. Using Pokémon as a case study, we focused on the complex interactions between type, stats, and movesets in turn-based battles. Our agent successfully generated new Pokémon capable of balancing a restricted pool from the original 151. A crucial component of this work was a dense numerical embedding that not only encoded stats and types but also battle dynamics. The framework used for generation is provided above.

Wave Function Approximation: Performant Level Generation for Games (2025)

My latest work, submitted to IEEE GEM, explores improving the performance of the WFC algorithm by reducing computation during the constraint satisfaction step. My approach involved using the marching squares algorithm for both input and output representation, alongside ML-based function approximation to iteratively generate each of the algorithm's constraint satisfaction step. These optimizations resulted in a 5× speed increase over the original algorithm.

Enhancing Player Interactions with LLM Driven NPCs (Ongoing)

I am currently supervising multiple undergraduate thesis projects in Computer Science and Software Engineering. One CS thesis project is exploring how to improve NPC dialogue interactions by using Large Language Models (LLMs) in a closed system. Traditional LLM-driven NPCs often allow unrestricted input, leading to immersion-breaking responses. This research proposes having the LLM generate player prompts, ensuring all interactions remain consistent with the game’s setting. Additional work investigates NPC memory storage, retrieval, and the maintenance of social structures to inform NPC behavior and dialogue.
